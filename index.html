Quem sou

Profissional de dados com visão de negócio

Sou o Daniel — analista de dados em formação, focado em construir soluções que resolvem problemas reais do ambiente operacional. Atuo na interseção entre tecnologia e negócio, transformando dados dispersos em dashboards que gestores realmente usam para decidir.

Meu ecossistema principal é o Microsoft Power Platform: Power BI com DAX, transformações com Power Query, automações via Power Automate e apps com Power Apps. Cada projeto começa com um problema concreto e termina com impacto mensurável.

Estou em crescimento contínuo — estudando modelagem de dados, análise estatística e boas práticas de BI para entregar mais valor a cada projeto.
